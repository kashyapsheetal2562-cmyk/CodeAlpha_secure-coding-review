# 🔒 Secure Coding Review

## 📌 Overview
This repository contains a **Secure Coding Review** project focused on auditing application code for potential vulnerabilities.  
The goal is to demonstrate how to identify insecure coding practices, document findings, and provide recommendations for safer software development.

---

## 🛠️ Methodology
- Manual inspection of source code (`src/` folder).
- Static analysis using tools like **Bandit** (Python), **SonarQube**, or similar.
- Review against **OWASP Secure Coding Guidelines**.
- Documentation of findings in `SECURE_REVIEW.md`.
